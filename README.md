# GoodWe ChargeOps Assistant

## Descrição do Projeto

O GoodWe ChargeOps Assistant é um chatbot inteligente desenvolvido para o GoodWe EV Challenge 2026.

O sistema auxilia operadores comerciais no monitoramento de carregadores elétricos através da interpretação de dados obtidos por APIs.

---

## Problema

Operadores precisam acessar múltiplos sistemas para acompanhar o status dos carregadores, aumentando tempo operacional e dificultando tomadas de decisão rápidas.

---

## Objetivo

Criar um chatbot capaz de:

- Consultar status de carregadores
- Interpretar informações operacionais
- Responder perguntas em linguagem natural
- Auxiliar monitoramento em tempo real e histórico

---

## Persona

**Operador Comercial**

Responsável por:

- Monitorar carregadores
- Acompanhar utilização
- Apoiar decisões operacionais

---

- APIs

## Dados utilizados via API

- id_carregador

- potencia_atual
- energia_consumida
- horario_sessao
- usuario_conectado
